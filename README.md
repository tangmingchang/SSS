# SSS

影动未来：AI智能皮影戏数字化创作、演示与教学的综合平台。

## 项目结构

- `frontend/`：前端（React + Vite）
- `backend/`：后端（Flask）

## 本地运行

### 1) 启动后端

```bash
cd backend
pip install -r requirements.txt
python app.py
```

默认地址：`http://localhost:5000`

### 2) 启动前端

```bash
cd frontend
npm install
npm run dev
```

默认地址：`http://localhost:3000`

## 说明

- 仓库仅保留部署所需的代码与运行资源。
- `.env` 等敏感信息不纳入版本控制。

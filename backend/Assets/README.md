# Unity 皮影体感交互成品项目

本目录为 **皮影戏体感交互** 的 Unity 成品项目，用于**数字人/体感**展示与制作。

## 内容说明

- **PiYing**：皮影主场景与 UI（Start、Main、PiYing 等场景）
- **Kinect**：K2Examples 提供 Kinect 体感驱动（骨骼、交互等）
- **Prefabs**：皮影角色预制体（左右角色 zuo1/zuo2、you1/you2 等）

## 使用方式

1. 使用 **Unity** 打开本目录（或包含本 Assets 的 Unity 工程）。
2. 运行 PiYing 相关场景，连接 Kinect 即可进行体感操控皮影。
3. 工作台生成的 **BVH** 可从 Web 端下载后，在 Unity 中导入使用（若项目内已接 BVH 播放逻辑）。

## 与 Web 工作台的关系

- Web 工作台负责：剧本生成、角色生成、动作捕捉（视频→BVH）、演出导出等。
- **数字人/体感交互** 以本 Unity 项目为准，不再使用原 Web 端的 BVH→数字人视频生成方案。
- **演出视频导出**：交付站导出视频时，若用户未上传场景图/舞台截图，后端会优先使用本目录 `PiYing/Mew` 下的默认场景图（如 `newbg.png`、`bbg.png`）作为视频画面，与念白+音乐合成 MP4。
